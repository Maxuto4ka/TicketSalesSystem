# TicketSalesSystem
# 🎟️ Ticket Sales System (OLTP & OLAP)

Десктопний WPF-додаток для автоматизації обліку продажів квитків на культурно-масові заходи та глибокого багатовимірного аналізу даних (Business Intelligence). 

Він демонструє повний цикл роботи з даними: від операційного введення (OLTP) до міграції (ETL) та візуалізації у сховищі даних (OLAP).

## 🚀 Основні можливості

### 📝 Операційний контур (OLTP)
* **Повноцінний CRUD:** Додавання, перегляд, редагування та видалення записів про продажі квитків.
* **Автоматизація:** Динамічний розрахунок загальної вартості (`Кількість × Ціна`).
* **Довідники:** Використання нормалізованих довідників (Заходи, Менеджери, Клієнти, Локації) через зручні випадаючі списки.

### 🔄 Екстракція та завантаження (ETL)
* Вбудований механізм міграції нових транзакцій з нормалізованої бази даних (3NF) до багатовимірного сховища даних (схема "Зірка").

### 📊 Аналітика та Дашборди (OLAP)
* **Багатовимірні зрізи:** Формування звітів за менеджерами, типами заходів, містами та часом.
* **Динамічна фільтрація:** Можливість фільтрувати дані "на льоту" (наприклад, вибір конкретного року для перебудови всіх графіків).
* **Інтерактивна візуалізація (LiveCharts):**
  * 📈 *Динаміка продажів у часі* (Лінійна діаграма)
  * 📊 *Виручка за місяцями* (Вертикальна гістограма)
  * 🏆 *Рейтинг менеджерів* (Стовпчикова діаграма, `ORDER BY DESC`)
  * 🏙️ *Топ міст за виручкою* (Горизонтальна діаграма)
  * 🥧 *Частка повернених квитків* та *Частка за типами заходів* (Кругові діаграми)

## 🛠️ Технологічний стек

* **Мова програмування:** C# (.NET)
* **UI Фреймворк:** WPF (Windows Presentation Foundation)
* **База даних:** Microsoft SQL Server (T-SQL)
* **Доступ до даних:** ADO.NET (`SqlConnection`, `SqlCommand`)
* **Візуалізація:** LiveCharts.Wpf
* **Тестування:** xUnit, Moq (Unit-тести з використанням мокування інтерфейсів БД та ізоляцією STA потоків)
